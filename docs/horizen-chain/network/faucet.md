The testnet faucet provides free test ETH for development and testing on Horizen Testnet.

Faucet URL: [https://hub-testnet.horizen.io/](https://hub-testnet.horizen.io/)

### To claim testnet ETH

- Connect your wallet on the faucet page.
- Ensure your wallet is set to Horizen Testnet (Chain ID: 2651420).
- Submit a claim. The funds arrive within a few seconds.

Testnet ETH has no real value and is only valid on the Horizen Testnet network. If you need additional testnet ETH beyond faucet limits, you can bridge Base Sepolia ETH to Horizen Testnet via the testnet bridge.
