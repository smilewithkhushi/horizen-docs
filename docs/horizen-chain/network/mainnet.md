Add the following network to your wallet or development environment to connect to Horizen Mainnet.

### Horizen Mainnet (Base)
| Parameter      | Value     |
| ------------------ | ---------------------------------------------------------------------------- |
| Chain ID | 26514 |
| RPC(HTTPS) | https://horizen.calderachain.xyz/http |
| RPC(WS) | wss://horizen.calderachain.xyz/ws |
| Symbol     | ETH |
| Block Explorer URL | https://explorer.horizen.io/ |
| Mainnet Hub | https://hub.horizen.io/ |

### Adding to MetaMask manually:
1. Open MetaMask → Settings → Networks → Add Network
2. Click Add a network manually
3. Fill in the values from the table above
4. Click Save - Horizen Mainnet is now available in your wallet
