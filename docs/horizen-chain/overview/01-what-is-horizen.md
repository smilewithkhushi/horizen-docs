# What is Horizen Chain?

Horizen is an EVM-native blockchain and privacy app ecosystem that rolls up to Base, designed to enable regulatory-compliant private onchain applications for businesses and privacy-minded retail users.

Built as an L3 on the OP Stack over Base, Horizen inherits Ethereum's security while offering privacy tooling that no other L2 provides natively. Developers can deploy standard EVM dApps on Horizen Chain and optionally use VELA, Horizen's confidential compute layer, to add verifiable privacy.

## Key properties

- **EVM-compatible** — Any Solidity contract, Hardhat or Foundry project works out of the box. No migration friction.
- **Built on Base** — Horizen rolls up to Base (L2) which settles to Ethereum (L1), inheriting full Ethereum security.
- **Privacy by design** — VELA adds confidential execution via Trusted Execution Environments (TEEs) to the EVM.
- **Regulatory-compliant** — Authority services allow selective disclosure for audits, KYC, and compliance workflows.
- **Low fees** — Inherits Base's low gas costs with additional compression from the OP Stack.

## What can you build?

- Private DeFi protocols (DEXes, lending, perpetuals)
- Confidential identity and KYC solutions
- On-chain gaming with hidden state
- Enterprise applications requiring auditability without full public disclosure
- Privacy-preserving stablecoins and payment rails
