# Horizen Overview

Horizen is an EVM-native blockchain and privacy app ecosystem that rolls up to Base, designed to enable regulatory-compliant private onchain applications for businesses and privacy-minded retail users. Additional privacy tooling from Horizen Labs is currently under development.
