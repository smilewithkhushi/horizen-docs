# VELA

VELA provides a Private Vault for Web3 Apps and is a chain-agnostic confidential coprocessor. Applications execute inside Trusted Execution Environments (TEEs), where every computation is cryptographically attested and no operator, competitor, or cloud provider can access the data. Regulators can verify compliance through cryptographic proof without requiring access to the underlying information.

## Limitations

Not yet deployed to production environments.

Limited to a local development setup using Docker.

TEE environment is emulated on local environment - Once you have a running prototype we can provide dedicated AWS nitro instances on Horizen chain for your team upon request.

Only one WASM application can be deployed into the environment.
